# Indian Roads Analytics Portfolio

## 📌 Project Overview
This portfolio demonstrates an end-to-end **data analytics pipeline** using **Databricks, PySpark, SQL, Power BI, and Tableau**.  
The dataset used is the **Indian Roads dataset**, which contains raw, uncleaned information about road lengths, conditions, traffic volumes, and accident counts across states and districts.

The project showcases:
- Data engineering skills (Bronze → Silver → Gold layers).
- Cleaning and transformation using PySpark.
- KPI design and SQL queries.
- Interactive dashboards across multiple BI tools.
- GitHub documentation for reproducibility.

---

## 🛠️ Tech Stack
- **Databricks Community Edition** (PySpark + SQL for ETL and pipeline design).
- **Power BI Free** (interactive dashboards).
- **Tableau Public** (visual storytelling).
- **GitHub** (portfolio hosting and documentation).
